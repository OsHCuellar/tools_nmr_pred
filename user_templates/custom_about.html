Web-tool for the analysis and visualization of the data of nmr_pred. Project for the "AI for chemistry" course of EPFL.

The current version only serves as a tool to analyze the chemical diversity. The prediction is not implemented yet.

In the chemical diversity section you have to select first the latent space, and then one of the possible options. Each of these options contains implicitly information about the representation ({rep}), metric ({metric}), perplexity ({perp}) and target/label ({target}).
